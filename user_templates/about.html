Program that interprets .cif files of molecular crystals and retrieves structural, connectivity and charge information of all molecules present in the unit cell.

This includes (but is not limited to):

  • Atomic coordinates, labels
  • Total molecular charge, and formal atomic charges, including metal oxidation states
  • Connectivity network as defined by either the adjacency matrix or the bond-order matrix

This tool is based on the cell2mol package by LCMD at EPFL, adapted via the tools-barebone framework.

You can find a full list of contributors in the README of the GitHub repository.

You can find the source code of this tool on the GitHub repository materialscloud-org/tools-cell2mol.
